This app allows users to manage their tasks in a simple and efficient way by adding, checking off, and deleting tasks.

Description:

The Todo List app allows users to:
Add tasks: Users can type in a task in the input field and click the "ADD BUTTON" to add it to their to-do list.

Delete tasks: Users can delete tasks from their to-do list by clicking the delete button beside each task.

Check off tasks: Users can mark tasks as completed by clicking on the checkbox. Once checked, the task will move to the "Completed Task" section.

Delete completed tasks: Users can also delete tasks from the "Completed Task" list if needed.


Features:

Add Task: Type your task title and click "ADD BUTTON."

Todo List: All tasks you add will appear here.

Delete Task: Each task has a delete button to remove it from the list.

Checkbox: Tasks can be marked as completed by checking the checkbox.

Completed Tasks: Once checked, tasks will move to the completed section.

Delete Completed Task: You can delete tasks from the completed section as well.



Instructions:

To Add a Task:
Enter the task title in the input box.
Click the "ADD BUTTON" to add it to your to-do list.


To Delete a Task:
Click the delete button next to any task in either the "Todo List" or "Completed Task" sections to remove it.


To Mark a Task as Completed:
Click the checkbox beside the task, and it will move to the "Completed Task" section.


To Delete a Completed Task:
Tasks in the "Completed Task" section can also be deleted by clicking the delete button.
